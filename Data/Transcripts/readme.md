## Transcripts Data

Lecture Transcripts data is created by extracting English lecture transcripts and translating them into Hindi and Bengali using Google Translate.

We considered lecture transcripts from the NPTEL website (CS and AI domains).

Due to the file size limitations, we have uploaded data externally, which can be accessed at: [GDrive Link](https://drive.google.com/drive/folders/1e4PeUZpLFj4az2JDpQZA37T5CXcbaJMi?usp=sharing)
